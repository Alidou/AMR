Release

Version 1.5

From this version, All Mangas Reader is open source (licence has not been yet chosen) and source code is hosted on GitHub.

This will make AMR evolve faster ! Anyone can fork All Mangas Reader's code and fix issues. If you want to participate, ask to enter the developer team on the forum !

AMR is now automatically packaged from GitHub code and the release process is lighter than before. We should be able to produce more versions and to fix bugs faster.

A beta channel has been created to allow anyone to have the latest fix in their AMR. If you use the beta channel. Your AMR will be updated each time a commit is made on GitHub. You should encounter bugs, regression but you will help us a lot by being able to declare new issues on GitHub !
Note that if you are on the beta channel, you will have to uninstall AMR before returning to stable channel. Both versions can be downloaded from AMR's main website.

The purpose of AMR V1.5 was to migrate the extension to Manifest V2. Google Chrome changed their security policy and whole parts of the extension were not compliant with these requirements. We had to recode a lot of things...

About malware warnings

Some of you have reported Malware warnings from AMR. The problem is that AMR access the web site www.thespectrum.net which is detected as containing malware by Avast and other free antiviruses... On thespectrum web site, in the FAQ, you can read this :

Why is my antivirus saying your site is bad ? I've received this question a few times and the bottom line is my website does NOT have viruses. I've received these questions from people using Internet Explorer 6, easily one of the most virus-prone browsers out there. I've used countless virus-free computers to access this website and no antivirus program has ever told me my site was infected. I believe that those emailing my asking why my site says it's infected already have pre-existing viruses on their computer because they haven't taken the measures to protect their computer. Every computer needs to either have the updated Internet Explorer 7, or updated versions of other browsers such as Firefox and Opera. Users, especially those using Windows XP, should do some research and get good programs like ESET Smart Security, which is better than the numerous free antivirus programs out there.

This alert is a false positive ! Ignore it since antiviruses fix their virus base... If you are not sure of this web site, desactivate TheSpectrum web site in the option page, AMR will then not access this web site anymore.

As I received many requests, I added the following features :

  • Added an option to prevent images from being resized to fit the page. If you have a small resolution, uncheck this option so the scans won't be resized
  • Added an option to change the background color of the popup. As some of you prefered the black background, they can choose to come back to it.
  • Added an option to load scans in the order of the chapter. If you check it, the first scan will be loaded first then the second and so on. Checking this option will slow down the chapter loading but it is indicated if your connection is slow. In this case you don't have to wait too much time to start reading your chapter.
  • Added a delete button on the "rate the extension" panel in mangas web site. Clicking on this button will automatically uncheck the associated option so you won't have this panel anymore.
  • Fixed the problem with SomeManga
